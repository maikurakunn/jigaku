マインクラフト executeコマンド

executeコマンドは、様々な条件に基づいて別のコマンドを実行させる強力なコマンドです。

基本的な構文

execute as <エンティティ> at <座標> run <実行するコマンド>
  • as <エンティティ>: コマンドを実行するエンティティを指定します。
  • at <座標>: コマンドを実行する座標を指定します。
  • run <実行するコマンド>: 実際に実行するコマンドを指定します。

よく使うサブコマンド

  • if / unless: 特定の条件を満たす場合にのみ後続のコマンドを実行します。
    • if block <座標> <ブロック>
    • if entity <エンティティ>
    • if score <エンティティ> <目的> <演算子> <値>
    • if data <ストレージ> <パス>
    • unless は if の否定形です。
  • positioned <座標>: コマンドを実行する座標を変更します。
  • rotated <ヨー> <ピッチ> / rotated as <エンティティ>: コマンドを実行するエンティティの向きを変更します。
  • dimension <ディメンション>: コマンドを実行するディメンションを指定します。
  • anchored <足 | 目>: コマンドの基準点をエンティティの足元または目に設定します。
  • store <結果の種類> <ターゲット> <データパス> <スケール>: コマンドの結果を保存します。

便利な応用例

  1. 特定の場所にいるプレイヤーにメッセージを表示する: 
    execute as @a at 100 64 200 if entity @s[distance=..5] run say こんにちは！近くにいますね！
  2. 特定のアイテムを持っているエンティティの周囲にパーティクルを表示する: 
    execute as @e[nbt={Inventory:[{id:"minecraft:diamond_sword"}]}] at @s run particle minecraft:flame ~ ~1 ~ 0.2 0.2 0.2 0.01 10 force
  3. 体力が一定以下のプレイヤーに回復効果を与える: 
    execute as @a[scores={health=..10}] run effect give @s minecraft:regeneration 1 1 true
  4. 特定のブロックを破壊したプレイヤーの近くに雷を落とす: 
    execute as @a[nbt={SelectedItem:{id:"minecraft:diamond_pickaxe"}}] at @s if block ~ ~-1 ~ minecraft:stone run summon minecraft:lightning_bolt ~ ~ ~
  5. 特定の場所にいるエンティティの向きに合わせて矢を発射する: 
    execute as @e[type=minecraft:skeleton,tag=archer] at @s rotated as @s run summon minecraft:arrow ^ ^1 ^1 {Motion:[0.0,0.5,1.0f]}

注意点

  • 複雑になりすぎるとデバッグが難しくなることがあります。
  • セレクター、NBTデータ、スコアボードの理解が重要です。
  • コマンドブロックでの連続実行時の負荷に注意が必要です。
